Welcome to IP Range finder!

Use this tool to parse your effective routes and Dump Routing Info files and find which range is maching for a specific IP

To parse the Dump Routing Info file follow these steps:

  1. Go to the ExpressRoute page on ASC and download the file from the "DumpRoutingInfo" link
  2. Here on the tool, click on "Choose File" button on the first slot and upload the DumpRoutingInfo file
  3. On the first slot textbox type in the IP for which you want to find the maching range
  4. Click on the "Upload Dump Routing Info file" button to submit and parse the file
  5. You will then be directed to the output page with the result. If no range matched the IP you will get an empty response.

To parse the Effective Routes file follow these steps:

  1. Go to either the NIC ASC page and get the NIC Effective Routes, or to the VM Diagnostics blade on ASC and run a test traffic to get the NIC Effective Routes
  2. Once you have the results for the effective routes you need to download the Excel file from ASC
    • To do so, click on the "Export to Excel file" (first button on the top right corner of the table)
  3. Here on the tool, click on the "Choose File" button on the second slot and upload the Effective Routes Excel file
  4. On the second slot textbox type in the IP for which you want to find the maching range
  5. Click on the "Upload effective routes excel file" button to submit and parse the file
  6. You will be then directed to the output page with the result. If no range matched the IP you will get an empty response.
